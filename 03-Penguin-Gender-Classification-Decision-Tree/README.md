# Project 3: Penguin Sex Classification using Decision Trees

## What this project does
In this exercise, we worked on a classification task using the `penguins` dataset. The objective is to predict if a penguin is male or female based on its physical measurements.

## What I did:
* Handled categorical values (like species and islands) by encoding them into numbers using Pandas `.replace()`.
* Kept all dataset rows and columns to preserve data structure.
* Trained a **Decision Tree Classifier** from Scikit-Learn to make predictions based on measurements like bill size and flipper length.
* Generated a classification report to check the prediction accuracy.

## Libraries used:
* Scikit-Learn
* Pandas & NumPy
* Seaborn
